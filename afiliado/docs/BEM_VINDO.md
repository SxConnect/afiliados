## 💡 Dicas Importantes

### ⚠️ Antes de Começar
1. Certifique-se de ter a PAPI API instalada
2. Configure o arquivo `.env` com suas credenciais
3. Use Node.js 18 ou superior

### 🔐 Segurança
- Nunca commite o arquivo `.env`
- Use chaves criptográficas reais em produção
- Gere JWT secrets fortes

### 📊 Performance
- Use Redis para cache em produção
- Configure rate limiting adequado
- Monitore uso de recursos

## 🎓 Aprenda Mais

### Conceitos Importantes

**JWT (JSON Web Token)**
- Autenticação stateless
- Expira em 24 horas
- Renovável via refresh token

**Fingerprint de Máquina**
- Identifica dispositivo único
- Previne uso em múltiplos dispositivos
- Baseado em hardware

**Assinatura Criptográfica**
- Valida integridade dos dados
- Usa RSA 2048 bits
- Chave privada apenas na VPS

**Rate Limiting**
- Previne abuso da API
- 100 requisições por 15 minutos
- Configurável por rota
